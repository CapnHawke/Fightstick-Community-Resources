# Fightstick-Community-Resources
A gathered list of resources, hyperlinks, and various files for Open Source repositories and 3d prints for Fightstick builders and tinkerers. Subjects will be organized broadly by topic with attribution provided. <br>

From time to time, this readme will be updated with a directory of files that have been gathered from various areas across the internet. It will include the following: 
- Knowledge and Information harvested from various sources
- Links to open source software
- 3d models with attribution, or links to 3d models if the terms of an open source license require it.
- PCB designs with attribution, or linkes to PCB designs if the terms of an open source license require it.
- CAD drawings for various fightstick related parts with attribution, or links to those CAD drawings if the terms of an open source license require it.
- Full controller build plans with attribution, or links to those build plans if the terms of an open source license require it.

# Disclaimer
This information is being curated without specific endorsement and many of these files have not been tested by the curator. Use at your own risk. Happy modding!
