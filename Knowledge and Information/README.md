# The Unofficial Slagcoin Archive
This website has been an indispensable tool for fightstick makers for over a decade. For people just breaking into the hobby, this website contains detailed information about various industry standard arcade parts, as well as some information about how to hook arcade buttons and levers up to a variety of circuit boards that were not designed with arcade buttons and levers in mind. <br>

In 2026, Slagcoin was taken offline. Although many of the sections in the original document are beginning to feel dated. For example, it is becoming less and less common to hack commercial gamepads. The document also does not mention CAD programs, which are more frequently used nowadays to design control panels and enclosures. However, the foundational knowledge regarding electronics, soldering, electronics assembly and circuit board technology is still relevant. It can be viewed as a foundational roadmap and textbook for people learning how to make desired modifications to their controllers. It is a 101 level introduction to electronics work, and a master class in arcade parts and Fightstick assembly. <br>

The opinions expressed in Slagcoin regarding the quality and utility of certain components are the views of their respective authors and are not adopted or endorsed by this curator. Think for yourselves and form your own opinions! But form an educated opinion. <br>

Great thanks to GitHub user `nhamb/` who first archived Slagcoin seven years ago and made this possible. <br>

# How to Use This Archive

There are several different ways to access Slagcoin through GitHub. It will not be necessary to learn Python or any kind of code. First, you can visit nhamb's original archive of the site at the following Hyperlink: [Nhamb's slagcoin archive](https://github.com/nhanb/slagcoin-archive/tree/master) and download it directly. For the second method, you can download my repository and access it that way. Downloading my repository will also cause a bunch of other community resources to get downloaded.

For the first method, follow these steps.
1. Visit [Nhamb's slagcoin archive](https://github.com/nhanb/slagcoin-archive/tree/master).
2. Click on the bright green "<> Code" button to view a drop down menu.
3. Click "Download ZIP" and select a destination for your download.
4. When the Zipped download is complete, open it in a file explorer. On a windows PC, use the "Extract all" function to decompress the files. Or, in other operating systems, use whatever tools you have to extract files. Select a location on your local drive to save a copy of the website. 
5. Click on one of the HTML files in the decompressed Slagcoin archive. It should open in a web browser. You should now be accessing a fully functional version of the website that is saved locally on your own computer, with working hyperlinks that will take you to other parts of the archive.  

For the second method, follow the steps above, except replace Nhamb's slagcoin archive with my own repository, [Fightstick Community Resources](https://github.com/CapnHawke/Fightstick-Community-Resources). 

You can also clone either repository through GitHub desktop to save a version of it locally on your own machine.

After you have done either of these steps, you will be able to access Slagcoin locally through your own web browser even if you have no internet access.
