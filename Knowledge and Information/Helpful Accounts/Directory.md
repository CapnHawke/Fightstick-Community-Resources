\# Directory



In certain cases it is more useful to group resources together than it is to list out individual files. For example, a single user might upload six different styles of arcade button, or six different enclosures, or develop replacement cable doors for multiple controllers, or even develop multiple modifications for a single existing commercial controller. 



Hyperlinks to user accounts with lots of resources exist here and are sorted by topic, and then alphabetically within each topic. 



Topics include:



* Full Controllers 
* Custom PCB creation
* Firmware for development boards
* Custom Arcade Buttons
* Custom Arcade Levers
* Replacement Parts, Modifications, and Accessories
* Other topics to be added in future commits, if needed.



\# Full Controllers



This topic covers all facets of controller design. Where a page is listed here, they will not be listed in other categories even if they have contributed to custom PCB creation, Firmware, or other things.



* The Open Stick Community has provided numerous hardware designs which are accessible in their Hardware Repository. This includes full controllers. They have also shared what is arguably the most ubiquitous open source gamepad firmware in use today: GP2040-CE. The \[Open Stick Community GitHub page is here](https://github.com/OpenStickCommunity).
* Jfedor has provided numerous useful resources for DIY controller makers. He has written code for his own controllers as well as existing development boards, he had designed PCBs, and he has created enclosures using both his own PCBs as well as existing development boards. Jfedor's designs are available at his \[GitHub page](https://github.com/jfedor2). 
* Bad64 has provided multiple full enclosures for people to download and 3d print. Bad64's designs are available at two different Github pages. First is \[Bad64's original GitHub User page](https://github.com/bad64) and second is \[SuperBad64's GitHub User page](https://github.com/superbad64).
* SGF Devices has open-sourced full controllers including remixes of controllers hosted by the Open Stick Community. \[SGF Devices GitHub](https://github.com/sgfdevices)

\# Replacement Parts, Modifications, and Accessories



* Numerous repositories maintained by \[Karina a/k/a BolSadguy](https://github.com/BolSadguy?tab=repositories). These resources include modifications for the Madcatz Versus, as well as custom designed accessories for use on other controllers. Karina's work includes Boards useful for any controller that implements D-type panel mounts. They can be used for custom USB cables, or installing slide switches or buttons. Karina's work also includes modifications to existing arcade buttons such as custom Sunmulon SS plungers and modifications for the Qanba Gravity KS button line.
